## Getting Started

How to work with the repo

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/...
   ```
2. Install packages
   ```sh
   flutter pub get
   ```
3. Run app
   ```sh
   flutter run lib/main.dart
   ```
4. Run test
   ```sh
   flutter test
   ```
For help getting started with Flutter, view our online
[documentation](https://flutter.io/).
